# @collabai/hofos-ui

Built hofOS UI surface for CollaborationAI. Consume it from a GitHub Release tarball:

```json
"@collabai/hofos-ui": "https://github.com/jhoetter/collaboration-ai/releases/download/v0.1.0/collabai-ui-0.1.0.tgz"
```

Run `pnpm --filter @collabai/hofos-ui build:dist` to produce the library-mode `dist/` artifact.
